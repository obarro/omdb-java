# omdb-java
The OMDb API library for Java clients.

### Maven Dependency for Consumers ###
Add to your pom.xml:

```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
	<artifactId>omdb-java</artifactId>
	<version>1.0.1</version>
</dependency>
```
